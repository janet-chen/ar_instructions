Taxonomy of AR Instruction Primitives


Dimension Category Definition Examples
Representation Virtual Object Part(s) Virtual model of the object or a part of the object of assembly represented in any degree of realism, transparency, and size

via GIPHY

Virtual Tool Virtual model of tools represented in any degree of realism, transparency, and size
Virtual Indicator Virtual assets that are external to object parts and tools
Text Information communicated as words
Audio Information directed towards the senses of hearing of the user
Embodied gesture Representation of human or part of a human with a virtual model
Embedded multimedia content External (non-AR) multimedia content that is embedded into the manual
Purpose Prescriptive Primitives that recommend specific actions and/or explicit instructions to be followed.
Descriptive Non-prescriptive primitives that provide information, clarifications, explanations, etc.
Attentional Primitives that draw attention to specific points of interest.
Feedback Responses from the system (or from a virtual or remote assistant)
Instruction Placement World-stabilized Virtual assets anchored to part of user's environment
View-stabilized Virtual assets secured to screen and placement does not change with gaze
Instructor entity Real-time virtual assistance A software agent assisting the user
Real-time remote assistance A specialized worker in a separate location assisting the user
