Taxonomy


Instructions Interactions

Instructions

Virtual Object Part(s)

Virtual model of the object or a part of the object of assembly represented in any degree of realism, transparency, and size

Virtual Indicator

Virtual assets that are external to object parts and tools

Text

Information communicated as words

Audio

Information directed towards the senses of hearing of the user

Embodied gesture

Representation of human or part of a human with a virtual model

Embedded multimedia content

External (non-AR) multimedia content that is embedded into the manual

Prescriptive

Primitives that recommend specific actions and/or explicit instructions to be followed.

Descriptive

Non-prescriptive primitives that provide information, clarifications, and explanations.

Attentional

Primitives that draw attention to specific points of interest.

Feedback

Responses from the system (or from a virtual or remote assistant)

World-stabilized

Virtual assets anchored to part of user's environment

View-stabilized

Virtual assets secured to screen and placement does not change with gaze

Real-time virtual assistance

A software agent assisting the user

Real-time remote assistance

A specialized worker in a separate location assisting the user

 

Interactions

Tangible User Interface

Tangible devices that communicate from the user to the system

Voice Command

User's direction of sight gives input to the system

Gaze

Information communicated as words

Spatial Gesture

User makes a spatial movement to give input to the system

Head-worn display

Hardware worn on the head of the user

Handheld display

Portable hardware held by the user

Spatial Display

Hardware fixed statically within the environment
