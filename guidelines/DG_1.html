Implement the AR manual for display hardware that can fit the assembly workspace within its FoV to reduce excessive head movements for HMD and panning operations for HHD . If assembly workspace is larger than FoV, add visual signifiers of “offscreen” virtual assets.

 

Example

Support hands-free interaction techniques (e.g. voice commands) to free up the users hands.

 

Play video example

Shorten or replace verbose audio instructions with visual references when explaining specific object configurations.

 

Example

 

Supplement audio instruction with visual aids (e.g. text subtitles) when audio-only communication is not effective (e.g. for tasks that take place in a noisy environment)

 

Play video example

Only present virtual counterparts of the assembly object that are relevant for the current step.

 

Example

Directly superimpose virtual information on the real world for convenient, hands-free instructions, but be wary of occlusion.

 

Example

Rather than using static 2D diagrams, use 3D elements positioned in the user’s environment (e.g. superimposed virtual tools) to depict assembly actions.

 

Example

 

Rather than using abstract entities (e.g. arrows) to depict specific hand maneuvers, show a virtual hand or embed a demo video demonstrating the procedure so that users can easily copy the gestures.

 

Example

Use salient attentional visual indicators (e.g. high contrast color highlights) to help users quickly detect and identify targets in the real environment.

 

Example

View-stabilize critical information so that it is visible to the user regardless of where the device is pointed.

 

Example

 

Only use text when strictly required (e.g. labelling objects for training). Keep instructions short and simple.

 

Example

Include a visual representation of the remote assistant.
