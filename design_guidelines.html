Only present virtual counterparts of the assembly object that are related to the current step so that users can easily differentiate relevant information.

 

Example of virtual object parts

Directly superimpose virtual information on the real world for convenient, hands-free instructions, but be wary of occlusion.

 

Example of virtual object parts

Rather than using static 2D diagrams, use 3D elements positioned in the user’s environment (e.g. superimposed CAD models of virtual tools) to depict assembly actions.

 

Example of virtual object parts

 

Rather than using abstract entities (e.g. arrows) to depict specific hand maneuvers, show a virtual hand or embed a demo video demonstrating the procedure so that users can easily copy the gestures.

 

Example of embodied gestures

Use salient visual cues (e.g. high contrast color highlights) to help users quickly detect and identify targets in the real environment.

 

Example of visual indicators

View-stabilize critical information so that it is visible to the user regardless of where the device is pointed.

 

Example of view-stabilization

 

Include names of objects and tools for complex tasks to reduce the user's need to memorize them

 

Example of descriptive text

 

Only use text when strictly required. Keep instructions short and simple.
